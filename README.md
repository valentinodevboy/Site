# Site da Júlia

Projeto inicial feito em React + Vite, pensado para rodar no GitHub Codespaces e publicar na Vercel.

## Rodar no Codespaces

```bash
npm install
npm run dev
```

Depois abra a porta `5173` na aba **Ports/Portas** do Codespaces.

## Deploy na Vercel

1. Faça push do projeto para o GitHub.
2. Entre na Vercel.
3. Importe o repositório.
4. Framework: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.

## Onde trocar textos

Edite `src/main.jsx`.

## Onde trocar visual

Edite `src/style.css`.

## Onde trocar fotos

Coloque novas imagens em `public/fotos` e atualize a lista `photos` em `src/main.jsx`.
